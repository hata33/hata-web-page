# 语言切换器组件

## 功能描述
- 支持日语/中文双语切换
- 响应式设计，适配各种屏幕尺寸
- 全局状态管理，跨页面保持语言设置
- 本地存储持久化用户选择

## 技术实现
- **组件位置**: `src/components/ui/LanguageSwitcher.tsx`
- **状态管理**: `src/lib/language-context.tsx`
- **存储方式**: localStorage
- **使用方式**: 在 Navigation 组件中引用

## 注意事项
- 移动端显示在外层，不要放在菜单内
- 大屏幕显示在导航栏右侧
- 使用 localStorage 持久化语言设置
- 自动检测浏览器语言作为默认值
- 避免在移动端和桌面端重复显示
